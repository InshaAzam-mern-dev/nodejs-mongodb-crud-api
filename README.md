# nodejs-mongodb-crud-api
Basic CRUD API using Node.js, Express, and MongoDB (Mongoose)

# 🚀 Node.js MongoDB CRUD API

This project is a simple RESTful API built using Node.js, Express, and MongoDB. It demonstrates basic CRUD operations using Mongoose.

# 📌 Features

* Create a user (POST)
* Get all users (GET)
* Update user (PUT)
* Delete user (DELETE)

# 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose

# 📂 Project Structure

models/
routes/
config/
app.js

# ▶️ Run the Server


node app.js


# 🔗 API Endpoints

| Method | Endpoint   | Description   |
| ------ | ---------- | ------------- |
| POST   | /users     | Create user   |
| GET    | /users     | Get all users |
| PUT    | /users/:id | Update user   |
| DELETE | /users/:id | Delete user   |
