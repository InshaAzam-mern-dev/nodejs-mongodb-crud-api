# 🤝 Contributing Guide

Thank you for your interest in contributing to this project! 🎉

We welcome contributions from everyone — whether you're fixing a bug, improving documentation, or adding new features.

---

## 📌 How to Contribute

1. **Fork the repository**
2. **Clone your fork**

```bash
git clone https://github.com/your-username/nodejs-mongodb-crud-api.git
```

3. **Create a new branch**

```bash
git checkout -b feature/your-feature-name
```

4. **Make your changes**

5. **Commit your changes**

```bash
git commit -m "Added: your feature description"
```

6. **Push to your branch**

```bash
git push origin feature/your-feature-name
```

7. **Create a Pull Request**

---

## 🛠️ Contribution Ideas

You can contribute by:

* Improving code structure
* Adding validation
* Enhancing error handling
* Writing better documentation
* Adding new API features

---

## 📋 Guidelines

* Write clean and readable code
* Follow proper naming conventions
* Keep commits clear and meaningful
* Test your changes before submitting

---

## 💬 Questions?

If you have any questions, feel free to open an issue.

---

Thank you for contributing! 🚀
